# SaaS Analytics Dashboard UI Design Assets

## 1. Design Concept: Neon-Dark High Contrast
The design adopts a "Cyberpunk Minimal" aesthetic, utilizing a deep Slate 900 background to reduce eye strain while employing high-saturation neon accents for visual hierarchy and data emphasis.

### Color Palette
- **Base Background**: #0F172A (Slate 900)
- **Surface/Card**: #1E293B (Slate 800)
- **Primary Accent**: #38BDF8 (Electric Blue)
- **Success/Growth**: #4ADE80 (Neon Green)
- **Alert/Error**: #F472B6 (Neon Pink)

### Typography
- **Font**: Inter (sans-serif)
- **Optimization**: Tabular numbers for data alignment, tight letter-spacing for headings.

## 2. Layout & Navigation
- **Collapsible Sidebar**: Maximizes workspace. Uses a push-type behavior on desktop and overlay on mobile.
- **Bento UI Grid**: A 12-column grid system that organizes widgets into distinct cards, ensuring clarity in complex data environments.
- **Top Navigation**: Features global search (Cmd+K) and user settings with a backdrop blur effect.

## 3. Component Specifications
- **Interactive Line Chart**: Features 2px neon lines with glow effects and gradient area fills.
- **Density Heatmap**: Uses a dark-to-neon color scale to represent activity density intuitively.
- **Sortable Data Table**: Optimized for readability with hover highlights and fixed headers.
